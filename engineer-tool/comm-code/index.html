통신 코드 계산기

RS-232C, RS-485 등 시리얼 통신 프로토콜 구현 시 필요한 SUM, XOR 체크섬 및 Modbus CRC-16 값을 즉각 계산합니다. HEX 또는 ASCII 형태로 바이트 배열을 입력하여 기대되는 체크섬 및 CRC-16 코드를 생성하고 통신 프레임 오류를 검출하는 검산에 활용하세요.

대기
  • 기본 CRC: Modbus (poly 8005, init FFFF, refin/refout ON, xorout 0000).
  • 직접 입력: 직전 프리셋 값으로 열고 수정. 다시 프리셋 선택 시 잠금·값 덮어씀(확인창 없음).
  • LE: 2바이트 결과 저바이트 먼저 — Modbus RTU 전송 순서. WORD는 항상 고→저 표기.
  • 골든: ASCII 123456789 + CRC-16/Modbus → WORD 4B37 (LE 37 4B).
  • 입력 상한: HEX·ASCII 모두 최대 4096 bytes까지 계산하며 초과분은 무시하고 경고합니다.
