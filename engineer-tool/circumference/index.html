지름, 반지름, 원주, 면적 중 하나를 입력하면 나머지 3개의 값이 즉시 산출됩니다. 기구 설계 및 셋업 현장에서 PCD(기어) 탭을 활용해 기어 모듈과 잇수로 피치원 지름을 계산하고, 롤러나 턴테이블의 이송량을 손쉽게 환산할 수 있습니다.

원둘레 계산기

하나의 값을 입력하면 나머지 3개가 자동 산출됩니다 (mm / mm²)

◆원 도식
▶값 입력 (하나만 편집)지름 · 반지름 · 원주 · 면적 상호 변환
C = π × d  (= 2π × r)
A = π × r²  (= π × d² / 4)
⚠ 0 이하의 값은 입력할 수 없습니다

모듈(m)과 잇수(z)를 입력하면 피치원 지름(PCD)이 자동 산출됩니다 (단방향 — 역산 없음)

◆기어 도식
▶모듈 · 잇수 입력PCD = m × z (단방향, 역산 없음)
PCD = m × z
⚠ 모듈은 0보다 크고 잇수는 1 이상 정수여야 합니다
  • 변환 — 지름·반지름·원주·면적 중 마지막으로 편집한 값이 입력이 되고 나머지 3개는 자동 산출됩니다(단위 mm 고정, 면적 mm²). 값이 비면 입력 필드를 제외한 나머지가 비워지고, 0 이하 입력은 오류로 표시됩니다.
  • 도식 — 원 도식은 지름선(d)·반지름선(r)·원주 강조 호(C)·면적(A) 위치를 보여주는 참고도입니다. 기어 도식은 피치원(점선)과 톱니 수(z)에 반응하는 모식도입니다.
  • PCD 탭 — 모듈·잇수 → PCD 단방향 산출(역산 없음). 표시 값은 소수 4자리, 반올림 HALF_UP, 끝자리 0은 생략됩니다.
