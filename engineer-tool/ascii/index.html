ASCII 코드표 · 입력기

바코드 리더기, 비전 센서 등 시리얼 통신 장비의 데이터 프레임을 분석할 때 필수적인 0~127 표준 ASCII 코드표를 제공합니다. 상단의 입력기 탭에서는 문자열을 16/10진수 배열이나 PLC D 레지스터 워드값으로 손쉽게 상호 인코딩 및 디코딩할 수 있습니다.

ASCII 코드표

코드(10진/16진, 0x접두사 허용) 또는 문자 1자로 검색. 행을 탭하면 코드 정보를 복사한다.

▶검색코드(10진/16진) 또는 문자로 검색
DECHEXOCT문자내용
▶문자 → 코드
⚠ 비ASCII 문자가 포함되어 있습니다 (하위 8비트만 사용)
—
▼PLC 워드값문자 입력 기반 D 레지스터 패킹(2문자/워드)
—
◆코드(16진) → 문자
—
  • 범위 — 0~127의 7비트 표준 ASCII만 지원하며 확장 ASCII는 제외한다.
  • PLC 워드 — 2문자를 1워드로 묶고, 홀수 길이의 남는 바이트는 0x00으로 채운다. 하위 먼저(LH)가 미쓰비시 기본 순서다.
  • 16진 입력 — 연속 입력은 앞에서부터 2자리씩 1바이트로 읽는다. 잘못된 문자는 건너뛰고 경고한다.
