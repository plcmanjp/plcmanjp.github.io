파라미터(기준축 입력값 / 변환값 출력값)를 위에서 등록하고, 아래 현재값·변환값 존과 XY 전달 특성 그래프에서 PV를 바꿔가며 검산한다. 그래프 위를 좌우로 드래그하면 PV가 유동적으로 변한다.

아날로그 스케일 변환

분해능(디지털 값)을 기준으로 대응 신호와 공학값을 동시에 환산합니다.

▶기준축(입력값) 파라미터분해능, 전압/전류신호
▼변환값(출력값) 파라미터공학단위
◆현재값 · 변환값
변환값—
대응 신호—
SPAN_e = 0.0250   OFFSET_e = 0.0000
변환값 = SPAN_e × PV + OFFSET_e
신호 = SPAN_s × 분해능 + OFFSET_s
⚠ 입력 범위 초과 (외삽값)
전달 특성 그래프 (드래그로 PV 조정)
  • 축 — X=분해능(정의역) MIN~MAX 10분할 +−1밴드 · Y=변환값(공학값) MIN~MAX 5분할 +−1밴드. 전달선(정의 구간)·현재 PV 마커(점선 가이드).
  • 드래그 — 그래프를 좌우로 끌면 PV가 ±1밴드 범위까지 유동 변경(직접 입력 병행). 범위 밖은 외삽 + 주황 경고.
  • 단위 — 변환값 단위는 상시 입력창 + ▼(상용 프리셋 kPa·MPa·bar·psi·torr·µm·mm·m). 전압/전류 신호는 기본 꺼짐(전류 mA / 전압 V, 종류가 프리셋 필터).
