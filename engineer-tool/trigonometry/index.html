  • 3탭 — 삼각함수(각도↔라디안·정삼각·역삼각) / 직각삼각형(각도+기준변→변, 두 변→각도) / 좌표(두 점→방향각·거리, ATAN2). 각 탭은 독립 계산 — 서로 값을 되먹임하지 않는다.
  • 부호 전환(±) — 각도·라디안·좌표는 음수가 정상 입력이라 키보드 마이너스 키에 의존하지 않고 버튼으로 부호를 뒤집는다. 각도↔라디안은 내부에 raw 값을 보존해 반올림 재입력에 의한 정밀도 손실이 없다.
  • tan 특이점 — 90°+k·180°에서는 |cos|<1e-10 판정으로 "정의되지 않음"을 표시한다(거대 유한값 노출 금지). asin/acos 는 |x|>1 이면 도메인 오류.
  • 직각삼각형 정의역 — 각도는 0°<θ<90° 엄격(경계·범위 밖=오류), 기준변 값은 0보다 커야 한다. 역방향(두 변→각도)은 다리변이 빗변보다 작아야 성립한다.
  • 좌표 zero-vector — P1과 P2가 같으면 거리는 0이지만 방향각은 정의되지 않아 "두 점이 같음" 으로 표시한다(0° 오표시 방지).
