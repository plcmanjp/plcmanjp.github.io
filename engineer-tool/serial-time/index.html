통신속도는 1 이상의 정수여야 합니다. 프리셋에 없는 통신속도입니다. 실제 장치 지원 여부를 확인하세요.

이론상 선로 시간에 입력한 지연만 더한 값입니다. 실제 장치 응답시간과는 차이가 있을 수 있습니다.

  • 계산식 — 문자당 비트 = 1(시작) + 데이터 + 패리티(0/1) + 정지, 전송시간 = 총 비트 수 ÷ 통신속도.
  • 엄격 파서 — 홀수 니블, 비HEX 문자, 비ASCII 문자, 잘못된 이스케이프는 오류로 처리하고 결과를 비운다.
  • RS-485 전환 지연 — 2선식 반이중일 때만 총시간에 합산한다. 4선식·RS-232·RS-422은 미적용.
  • 검산값 — 9600bps·8N1·8바이트는 8.333 ms, 115200bps·8N1·64바이트는 5.556 ms다.
  • 입력 상한 — HEX·ASCII 모두 4096 bytes이며 초과 시 계산을 차단한다.
