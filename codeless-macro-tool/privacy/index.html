1. 개요

JP's Codeless Macro Tool(이하 "앱")은 개인정보를 수집하지 않습니다. 계정 생성, 로그인, 사용자 식별, 텔레메트리(사용 통계 수집), 광고 SDK가 앱에 포함되어 있지 않습니다.

2. 수집하는 정보

앱은 이름, 이메일, 위치, 기기 식별자 등 어떠한 개인정보도 수집·저장·전송하지 않습니다.

3. 네트워크 통신

앱이 인터넷에 접속하는 경우는 업데이트·변경내역 확인 한 가지뿐입니다.

  • 앱은 GitHub 공개 저장소에 있는 다음 두 파일을 HTTPS GET 요청으로 읽어옵니다.
    • https://raw.githubusercontent.com/plcmanjp/JPs-Codeless-Macro-Tool-Release/main/version.json
    • https://raw.githubusercontent.com/plcmanjp/JPs-Codeless-Macro-Tool-Release/main/release_notes/v{버전}.md
  • 이 요청은 표준 HTTP 요청이며, 앱이 사용자의 개인 데이터(이름, 이메일, 매크로 내용, 사용 기록 등)를 담아 보내지 않습니다.
  • 다만 이 요청이 GitHub 서버에 도달하는 과정에서 IP 주소 등 표준 접속 정보가 발생할 수 있으며, 이는 GitHub의 개인정보처리방침이 적용되는 영역입니다. 자세한 내용은 GitHub 개인정보처리방침에서 확인할 수 있습니다.
  • 업데이트 확인 외의 다른 서버 통신(로그인, 분석, 광고, 크래시 리포트 등)은 존재하지 않습니다.

4. 로컬 데이터 저장 위치

앱이 생성하는 모든 데이터(매크로 파일, 설정, 즐겨찾기, 실행 로그, 화면 캡처 이미지, OCR 인식 결과 등)는 사용자의 PC에만 저장되며 외부로 전송되지 않습니다.

  • 포터블(ZIP) 배포판: 실행 파일(JPsCodelessMacroTool.exe)과 같은 폴더
  • Microsoft Store 배포판: %LOCALAPPDATA% 및 사용자 문서 폴더 하위 앱 전용 경로

이 데이터는 사용자가 직접 삭제하거나 앱을 제거할 때 함께 정리할 수 있으며, 개발자는 이 데이터에 접근할 수 없습니다.

5. 화면·입력 기능의 로컬 처리

앱의 주요 기능(마우스·키보드 입력 시뮬레이션, 화면 캡처, 이미지 검색, Windows 내장 OCR을 이용한 텍스트 인식)은 전부 사용자 PC 내부에서만 처리되며, 캡처된 이미지나 인식된 텍스트가 외부 서버로 전송되는 경로는 없습니다.

6. 아동 관련 고지

앱은 특정 연령층, 특히 아동을 대상으로 설계되지 않았으며, 아동으로부터 개인정보를 의도적으로 수집하지 않습니다.

7. 제3자 제공

앱은 어떠한 개인정보도 수집하지 않으므로 제3자에게 판매하거나 공유하지 않습니다.

8. 방침 변경

본 방침은 앱 기능 변경에 따라 갱신될 수 있습니다. 변경 시 이 페이지의 시행일을 갱신하여 고지합니다.

1. Overview

JP's Codeless Macro Tool ("the App") does not collect personal information. The App does not include account creation, sign-in, user identification, telemetry (usage analytics), or advertising SDKs.

2. Information We Collect

The App does not collect, store, or transmit any personal information, including names, emails, location, or device identifiers.

3. Network Communication

The only time the App connects to the internet is to check for updates and changelogs.

  • The App reads two public files from a GitHub repository via HTTPS GET requests:
    • https://raw.githubusercontent.com/plcmanjp/JPs-Codeless-Macro-Tool-Release/main/version.json
    • https://raw.githubusercontent.com/plcmanjp/JPs-Codeless-Macro-Tool-Release/main/release_notes/v{version}.md
  • These are standard HTTP requests. The App does not include any personal data (name, email, macro contents, usage history, etc.) in these requests.
  • As with any HTTP request, GitHub's servers may process standard connection information (such as IP address) while serving the request; this falls under GitHub's own privacy policy, available at GitHub Privacy Statement.
  • The App makes no other server connections — no login, analytics, advertising, or crash-reporting services are contacted.

4. Local Data Storage

All data the App creates — macro files, settings, favorites, execution logs, captured screen images, and OCR recognition results — is stored only on the user's PC and is never transmitted externally.

  • Portable (ZIP) build: same folder as JPsCodelessMacroTool.exe
  • Microsoft Store build: under %LOCALAPPDATA% and the user's Documents folder, in app-scoped locations

Users can delete this data at any time, or remove it along with the App. The developer has no access to this data.

5. Local Processing of Screen and Input Features

Primary features of the App — mouse/keyboard input simulation, screen capture, image search, and text recognition via Windows' built-in OCR — are processed entirely on the user's PC. There is no code path that transmits captured images or recognized text to an external server.

6. Children's Privacy

The App is not directed at any specific age group, including children, and does not knowingly collect personal information from children.

7. Third-Party Sharing

Since the App does not collect any personal information, it does not sell or share any data with third parties.

8. Changes to This Policy

This policy may be updated as App features change. Any change will be reflected by updating the effective date on this page.
